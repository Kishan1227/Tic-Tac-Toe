A simple Tic Tac Toe game built in Python where two players compete using X and O.

Only standard python libraries used.

This project implements the classic Tic Tac Toe game in Python. Two players take turns placing their marks X or O on a 3×3 grid. The goal is to get three marks in a row (horizontally, vertically, or diagonally).

The game runs in the terminal and continues until one player wins or the game ends in a draw.
